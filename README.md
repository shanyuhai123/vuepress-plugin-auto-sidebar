# Vuepress Plugin Auto Sidebar

## 介绍（Introduction）

这是为 vuepress 自动~~生成侧边栏~~**分组**的插件。



## 安装（Install）

```bash
npm i vuepress-plugin-auto-sidebar -D
```



## 使用（Usage）

```js
module.exports = {
  plugins: {
    "vuepress-plugin-auto-sidebar": {}
  }
}
```

该文档更多的使用说明见 [vuepress 自动生成侧边栏](https://shanyuhai123.github.io/vuepress-plugin-auto-sidebar)。
